Fonctionnalités

Tout ce dont vous avez besoin

Un outil complet, simple à utiliser, sans aucune configuration requise.

📝

Mode Texte

Collez directement votre texte dans la zone de saisie. Un compteur en temps réel vous indique le nombre de caractères saisis.

📁

Mode Fichiers

Glissez-déposez un ou plusieurs fichiers texte simultanément. Les doublons sont automatiquement détectés et ignorés.

✂️

Découpe Intelligente

L'algorithme respecte les mots et les phrases lors de la découpe. Aucun mot n'est coupé en plein milieu.

⚡

Découpe Brute

Coupe exactement à la limite définie, sans compromis. Idéal pour les cas où le respect strict de la limite est prioritaire.

📋

Copie en un clic

Copiez n'importe quelle partie dans le presse-papiers instantanément. Compatible avec tous les navigateurs modernes.

💾

Téléchargement .txt

Téléchargez chaque partie individuellement en fichier .txt, nommé automatiquement avec le nom de la source d'origine.

🔒

100% Privé

Tout le traitement se fait dans votre navigateur. Aucune donnée n'est envoyée sur un serveur. Votre contenu reste confidentiel.

📊

Résumé détaillé

Gestion des erreurs

Les fichiers vides ou illisibles sont signalés clairement sans interrompre le traitement des autres fichiers valides.

Fonctionnement

Comment ça marche ?

En quelques étapes simples, découpez n'importe quel texte ou fichier.

1

Choisissez votre source

Sélectionnez l'onglet 📝 Texte pour coller votre texte directement, ou l'onglet 📁 Fichiers pour importer un ou plusieurs fichiers. Vous pouvez combiner les deux : le texte collé et les fichiers seront tous traités ensemble.

💡 Astuce : vous pouvez glisser-déposer vos fichiers directement sur la zone de dépôt
2

Définissez la limite de caractères

Entrez le nombre maximum de caractères que doit contenir chaque partie. Par défaut, la limite est fixée à 10 000 caractères, idéale pour des outils comme ChatGPT. Adaptez cette valeur selon votre usage.

💡 Exemples : 1 600 pour les SMS, 4 000 pour les LLMs, 10 000 pour les emails
3

Choisissez le mode de découpe

Sélectionnez Intelligente pour une découpe respectant les mots et les phrases, ou Brute pour une coupure exacte à la limite définie. Le mode Intelligent est recommandé dans la grande majorité des cas.

4

Lancez la découpe

Cliquez sur ✂️ Découper le texte. L'outil traite immédiatement toutes les sources (texte + fichiers) et affiche les résultats organisés par source, avec le nombre de caractères et de parties générées.

5

Récupérez vos parties

Pour chaque partie générée, vous pouvez 📋 Copier son contenu dans le presse-papiers ou 💾 Télécharger le fichier .txt correspondant. Les fichiers sont nommés automatiquement (ex: mondocument_part1.txt).

💡 Une notification s'affiche brièvement pour confirmer la copie ou le téléchargement
Modes de découpe

Intelligent vs Brut

Deux algorithmes selon vos besoins.

⭐ Recommandé

✂️ Mode Intelligent

L'algorithme analyse le texte et cherche le meilleur point de coupure avant la limite : saut de ligne, espace, ponctuation. Le résultat est naturel et lisible.

  • Aucun mot coupé en plein milieu
  • Respect des paragraphes quand possible
  • Résultat propre et immédiatement utilisable
  • Idéal pour les textes narratifs, articles, emails
  • Parfait pour les LLMs et outils de traitement NLP
⚡ Précis

🔪 Mode Brut

La coupure est effectuée exactement à la position de la limite définie, sans aucun ajustement. Garantit que chaque partie ne dépasse jamais la limite.

  • Limite strictement respectée
  • Toutes les parties ont exactement la même taille
  • Traitement plus rapide sur très grands textes
  • Idéal pour les données structurées ou le code
  • Utile pour les systèmes avec limite absolue de bytes

Démarrage rapide

Comment l'utiliser

Aucune installation, aucune dépendance. Prêt en 10 secondes.

1

Cloner ou télécharger le projet

Récupérez les fichiers depuis GitHub

# Cloner le dépôt
git clone https://github.com/AlexandreTaboin/text-spliter.git

# Ou téléchargez le ZIP directement depuis GitHub
2

Ouvrir le fichier

Double-cliquez sur index.html pour l'ouvrir dans votre navigateur. C'est tout !

# Structure du projet
separateur-texte/
├── index.html    # ← Ouvrez ce fichier
└── README.md
3

Commencer à l'utiliser

Collez votre texte ou importez vos fichiers, définissez votre limite, cliquez sur ✂️ Découper et récupérez vos parties !
